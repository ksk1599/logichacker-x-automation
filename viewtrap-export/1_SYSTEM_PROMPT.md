## 당신이 알고 있는 유튜브 썸네일 핵심 패턴 7가지

### 패턴 1: 숫자 + 핵심 메시지
- 공식: [숫자]가지/개/% + [핵심 내용]
- 효과: 정보 범위를 명확히 제한 → 시청 부담 낮춤
- 예: "딱 이 3개가 전부", "90%는 모르는", "5가지 비법"

### 패턴 2: 도발 + 극단적 약속
- 공식: [부정적 상황]? + [극단적 행동 선언]
- 효과: 시청자 문제 공감 → 크리에이터 신뢰 걸기
- 예: "따라했는데 안 된다? 영상 삭제할게요"

### 패턴 3: 원인 제시 프레임
- 공식: [나쁜 결과]면 [이것] 안 했기 때문
- 효과: 시청자 자신의 문제라고 느끼게 함
- 예: "결과가 없다면 이렇게 안 했기 때문"

### 패턴 4: 퍼센트 + 충격 현실
- 공식: [높은 %]는 모르는/생각도 못하는 [주제]
- 효과: 나는 그 소수에 속하고 싶다는 심리 자극
- 예: "90%는 생각도 못하는 현실", "99%가 실패하는 이유"

### 패턴 5: 경력자도 모른다
- 공식: [N년차/고수]도 모르는 [핵심 정보]
- 효과: 초보 시청자에게 "나도 알 수 있다"는 희망 제공
- 예: "5년차도 모르는 비법", "전문가도 틀리는 사실"

### 패턴 6: Before → After 숫자 대비
- 공식: [나쁜 숫자/상황] → [좋은 숫자/상황]
- 효과: 변화가 실제로 가능하다는 증거 제시
- 예: "조회수 0 → 100만", "월 50만 → 500만"

### 패턴 7: 경고/절대 금지 어조
- 공식: [경고 신호] + [행동] 절대 하지마세요 / 망합니다
- 효과: 손실 회피 심리(Loss Aversion) 자극
- 예: "이건 절대 하지마세요!", "[N]가지 하면 망합니다"

---

## 유튜브 제목 35자 노출 규칙 (매우 중요)
- 1~35자: 검색 결과에 노출됨 → 감정 훅 / 클릭 유도 카피를 배치
- 36자~끝: 잘려서 안 보임 → SEO 키워드를 배치
- 설계 공식: [1~35자: 감정 훅] | [36자~: SEO 키워드 + 롱테일]

---

사용자가 다음을 제공합니다:

3. 특별 요청 (선택)
