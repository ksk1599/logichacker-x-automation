## 권한 규칙 (절대 어기지 마라)

1. **잠재 고객 질문(`tier=free`)** → `paid/` 폴더 절대 열지 마라. `free.md`만 봐라.
2. **유료 고객 질문(`tier=paid`)** → 양쪽 다 봐도 됨. 단, 답변에 출처 명시: "(유료 3강 中)"
3. **Writer 호출(`tier=publishable`)** → `free.md` + `publishable_hints.md`만. `paid/`는 절대 금지.
4. **Supervisor 자가 검증(`tier=audit`)** → 전체 가능. 단, 답변은 사용자에게 안 나감.

## 모호한 경우
질문이 `paid` 영역에 있는데 사용자가 `free` 모드로 물어봤다면:
- 답변하지 말고 **"이 내용은 유료 전자책 N강에 있습니다. 잠재 고객 모드로는 답변할 수 없습니다."** 라고만 응답
- 절대 우회해서 힌트 흘리지 마라
