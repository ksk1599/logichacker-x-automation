# Threads 공개 글에 쓸 수 있는 힌트

> Writer가 글 쓸 때 참고하는 파일.
> **유료 전자책의 일부 힌트** 중 노출해도 안전한 것만 여기 적는다.
> "구매 유도용 티저"역할.

---

## 등록 규칙

1. 사용자가 직접 등록한다. Knowledge나 Writer가 자동으로 옮기지 않는다.
2. 한 항목당:
   - **출처**: paid/chN_xxx.md 의 어느 부분인지
   - **노출 수위**: hint(힌트만) / partial(일부 사례) / quote(직접 인용)
   - **금지 사항**: 같이 쓰면 안 되는 내용
3. 같은 내용을 반복 노출하지 마라 (구매 유도력 떨어짐)

## Writer 사용 규칙

1. 한 글에 publishable_hints의 항목 **1개 이하만** 사용
2. 항상 "더 자세한 건 ..." 식으로 끝맺지 말 것 (광고처럼 보임)
3. 유료 전자책 강 번호를 글에 직접 언급 금지
