## 썸네일 문구 핵심 패턴 (채널 자체 패턴 7가지)

### 패턴 1: 숫자 + 핵심 메시지
- 공식: `[숫자]가지/개/% + [핵심 내용]`
- 예: "딱 이 3개가 전부", "90%는 생각도 못하는", "99%가 망하게 됩니다"
- 효과: 정보 범위를 명확히 제한 → 시청 부담 낮춤

### 패턴 2: 도발 + 극단적 약속
- 공식: `[부정적 상황]? + [극단적 행동 선언]`
- 예: "따라했는데 안 팔린다? 영상 삭제할게요"
- 효과: 시청자 문제 공감 → 크리에이터 신뢰 걸기

### 패턴 3: 원인 제시 프레임
- 공식: `[나쁜 결과]면 [이것] 안 했기 때문`
- 예: "매출이 없다면 이렇게 안 했기 때문"
- 효과: 시청자 자신의 문제라고 느끼게 함

### 패턴 4: 퍼센트 + 충격 현실
- 공식: `[높은 %]는 모르는/생각도 못하는 [주제]`
- 예: "90%는 생각도 못하는 스마트스토어 현실", "99%가 망하게 됩니다"
- 효과: 나는 그 소수에 속하고 싶다는 심리 자극

### 패턴 5: 경력자도 모른다
- 공식: `[N년차/고수]도 모르는 [핵심 정보]`
- 예: "5년차 셀러도 모르는 1페이지 상위노출 비법"
- 효과: 초보 시청자에게 "나도 알 수 있다"는 희망 제공

### 패턴 6: Before → After 숫자 대비
- 공식: `[나쁜 숫자] → [좋은 숫자]`
- 예: "배송전표 0건 → 115건", "2천원 → 1000만원"
- 효과: 변화가 실제로 가능하다는 증거 제시

### 패턴 7: 경고/절대 금지 어조
- 공식: `[경고 신호] + [행동] 절대 하지마세요 / 망합니다`
- 예: "이건 절대 하지마세요!", "스마트스토어 이렇게 하면 99%가 망하게 됩니다"
- 효과: 손실 회피 심리(Loss Aversion) 자극

## 도메인 지식 — 상세페이지 기획 원칙

> 로직해커 엑스가 영상에서 다루는 핵심 내용. 썸네일 제작 시 참고.

### 상세페이지 최상단 원칙
- **최상단이 가장 중요**: 고객이 상품 클릭 후 처음 맞이하는 화면
- **최상단 구성 요소**: GIF + 후킹 메시지
- **후킹 메시지 원칙**: 고객의 문제를 직접 겨냥한 카피라이팅
- **메타 적용**: 이 원칙은 썸네일 제작에도 동일하게 적용됨
  → 썸네일도 셀러의 문제를 직접 겨냥해야 클릭됨

### 타겟 셀러의 핵심 고통
1. 상품 클릭은 되는데 구매가 안 됨
2. 상세페이지를 보다가 고객이 나가버림
3. 첫 화면을 어떻게 구성해야 할지 모름
4. 상세페이지를 만들었는데 전환이 안 됨

## 썸네일 문구 작성 규칙

1. **글자 수**: 10~20자 이내 (2~3줄로 쪼개기)
2. **핵심어를 굵게/크게**: 썸네일에서 가장 큰 글씨가 핵심 메시지
3. **동사 마무리 선호**: ~하세요, ~됩니다, ~때문, ~할게요
4. **감탄사/강조어 활용**: 제발, 절대, 딱, 겨우, 드디어
5. **시청자 상황 직접 겨냥**: "안 팔린다?", "매출이 없다면"

### 데이터 증거 활용 시 추가 규칙
- 배경에 실제 데이터 화면 사용 시: 숫자를 텍스트로 명시 (예: "전환율 6% → 24%")
- "실제 광고 데이터" 또는 "실화입니다" 뱃지 추가 → 신뢰성 급상승
- 가장 충격적인 숫자 1개만 전면에 내세우기 (매출 or 전환율 중 선택)

## 유튜브 제목 작성 규칙

5. **검색 키워드 포함**: 스마트스토어 + [세부 주제] 조합

## 참고 영상 분석 시 체크리스트

외부 채널의 터진 영상을 분석할 때 아래를 파악한다:
1. 썸네일 문구: 어떤 패턴인가? (위 7가지 + 레퍼런스 A~D 중 어느 것?)
2. 감정 훅: 두려움/기대/호기심/분노/안도 중 어느 감정을 자극?
3. 대상 명시: 특정 대상(초보, N년차, 직장인 등)을 겨냥했나?
4. 숫자 사용: 수치가 있나? 어떻게 사용했나?
5. 2색 이상 텍스트 구조: 어떻게 색을 나눴나?
6. 로직해커 엑스 채널에 적용 가능한 변형 포인트는?
7. 이 파일의 "외부 레퍼런스 영상 분석" 섹션에 저장할 것
