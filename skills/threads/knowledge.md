# 로직해커 엑스 지식베이스 (Knowledge Agent 소스)

> ⚠️ **이 파일은 골격이다.** 사용자가 전자책 PDF 업로드 후 채워야 함.
> Knowledge Agent는 이 파일만 신뢰한다 — 정확하지 않으면 잘못 답한다.

---

## 1. 상위노출 공식

**상위노출 점수 = 적합도 × 인기도 × 신뢰도**

### 적합도
- 키워드와 상품명, 카테고리, 속성의 일치도
- (전자책에서 정확한 정의 인용 필요)

### 인기도
- **최근 30일 데이터만 반영**
- **7일 가중치 가장 높음**
- 클릭, 찜, 구매, 리뷰 작성 등이 점수에 반영
- **외부 유입(네이버 검색, 인스타그램)은 인기도 0점 처리** — 네이버쇼핑 내부 트래픽만 인정

### 신뢰도
- 판매자 점수
- 리뷰 신뢰도 (조작 의심 시 감점)
- (전자책 보강 필요)

---

## 2. 키워드 분류

### 배열고정 키워드
- **순서 고정.** 순서 바꾸면 다른 상품으로 인식
- 예: **삼양라면** ≠ "라면 삼양"
- (전자책에서 정의/예시 추가)

### 붙박이 키워드
- **반드시 붙여쓰기.** 띄우면 다른 검색
- 예: ___ (전자책에서 사례 인용)

### 조립형 키워드
- **자유 배치 가능.** 순서/위치 무관
- 예: **꽃게라면** — "라면 꽃게"도 같은 상품
- (전자책에서 정의 보강)

---

## 3. 전략 — 체급전략

**검색량 낮은 키워드부터 1위를 잡아라.**

1. 처음엔 메인 키워드 노리지 마라 (경쟁자 점수 너무 높음)
2. 롱테일 키워드로 1위 만들기 → 인기도 점수 누적
3. 점수 쌓이면 점진적으로 큰 키워드로 확장
4. **키워드 많이 넣으면 점수 분산** → 핵심 1~2개 집중

---

## 4. 흔한 오해 (이걸 글에 절대 쓰지 마라)

| 오해 | 진실 |
|---|---|
| 키워드 많이 넣을수록 좋다 | ❌ 점수 분산 |
| 외부 유입(인스타) 늘리면 상위노출 | ❌ 인기도 0점 |
| 배열고정 키워드 순서 바꿔도 됨 | ❌ 다른 상품으로 인식 |
| 인기도는 누적 점수다 | ❌ 30일만 반영 |
| 리뷰 많으면 무조건 신뢰도 ↑ | ❌ 조작 의심 시 감점 |
