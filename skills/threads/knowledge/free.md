## 들어가기 전 — 상위노출 되어도 매출 0원인 이유

### 곱하기의 함정
- 상위노출 로직은 더하기(+)가 아니라 **곱하기(×)**
- 상품 매력이 0이면 완벽한 로직 100을 곱해도 결과는 0
- 매력 없는 상품 + 기술적 상위노출 = 독: 고객이 들어왔다가 "살 이유가 없네" 하고 이탈 → 인기도 점수 깎임
- "기본기 없이 기술만 쓰면 스토어는 더 빨리 무너진다"

### 팔리는 상품의 5가지 전략적 요소

#### 1. 가격: 자존심이 아니라 '전술'이다
- "무조건 싸게 팔라"는 1차원적 이야기 X
- 목표는 '초반 시장 침투'
- 진입 시: 마진을 줄이더라도 경쟁사보다 매력적인 가격으로 클릭/판매 유도
- 상위노출 고정 + 리뷰 쌓이면 → 가격 정상화로 마진 확보
- 가격은 시기에 따라 유동적으로 움직이는 전술

#### 2. 최저가: 미끼를 이용한 '우회로'
- 고가/마진 중요 상품은 최저가 못 맞춤 → 그래도 '최저가 정렬' 유입 놓치면 안 됨
- **미끼 상품 전략**:
  - 예: 냉장고 본품이 아니라 '냉장고 탈취제'/'전용 클리너' 같은 부속품을 압도적 최저가로 등록
  - 상세페이지에 메인 상품(냉장고) 링크 삽입
  - 1단계: 최저가순 검색 고객이 미끼(부속품) 클릭
  - 2단계: 상세페이지 링크 타고 메인 상품으로 유입
  - 3단계: 판매 발생 시 네이버가 인기도(판매점수) 부여
- 시스템 맹점을 이용해 진짜 상품에 점수를 먹이는 고급 기술

#### 3. 퀄리티: '희소성'에 깃발 꽂기
- "무조건 국산이 좋다"는 뻔한 소리 X — '시장 상황'을 보라
- 경쟁사 90%가 중국산인 카테고리 = 기회
- 국산(Made in Korea) 들고 나오면 강력한 차별화 포인트(USP)
- '안전'과 '신뢰'를 구매하려는 소비층은 반드시 존재
- 핵심: 남들이 못 하는 것을 하는 것 = 상대적 우위

#### 4. 옵션: 다 잡은 물고기, 제발 놓치지 마라
- 네이버 랭킹 핵심 = **[클릭수 + 판매 실적]**
- 옵션 부족 → 클릭만 들어오고 판매 0 → 경쟁사에 점수 헌납
- 예시:
  - 나의 점수: 클릭(+1) + 판매(0) = 1점
  - 경쟁사 점수: 클릭(+1) + 판매(+10) = 11점
- 옵션은 '방어막'이자 '객단가 상승의 치트키'
- "신발 사는 김에 양말도?", "선물용으로 하나 더?" 같은 객단가 상승 유도
- 자본주의 법칙: 네이버는 더 많은 수수료를 안겨주는 셀러를 편애

#### 5. [핵심] 소싱의 법칙 — 감으로 싸우는 건 '도박'이다
- 비즈니스는 도박이 아니라 데이터 싸움
- **온풍기 시장 예시**:
  - 온풍기(대표): 검색량 15만 (가장 치열한 전쟁터)
  - 루메나(브랜드): 검색량 6.6만
  - 파세코(브랜드): 검색량 3.8만
- 이름 없는 중국산 온풍기로 15만 키워드 1등? 불가능 (계란으로 바위 치기)
- 루메나 위탁 소싱 → 시작부터 6.6만 수요 등에 업고 시작
- 파세코는 38,170만 가져옴 → 같은 노력해도 루메나 판매자에게 패배
- 외부 유입(SNS)으로 점수 확보? **불가능** — 네이버 공식: 외부 링크 유입은 쇼핑검색 랭킹 점수에 미반영

## 1장 — 네이버는 사람이 아니다: AI가 매기는 채점표

### 채점표의 정체
- 네이버는 감정 없는 알고리즘
- 평가 요소 단 3가지: **적합도 × 인기도 × 신뢰도**
- 우리가 보는 '네이버 랭킹순' = 이 3가지 점수의 총점 정렬
- 30분 만에 1개 키워드만 바꿔서 17일 신규 상품을 1만 검색 키워드 10위권으로 진입시킨 사례

### 핵심 통찰
- 신뢰도는 변별력 없음 (페널티만 안 받으면 누구나 만점)
- 결국 [적합도, 인기도] 둘의 싸움
- 신규 셀러는 '인기도(판매량)'가 없음 → **적합도에 목숨 걸어야 함**
- "적합도만 제대로 건드려도, 판매량 없이 경쟁자를 밟고 올라가는 기적을 맛본다"

## 1장 — ① 적합도 가점: 검색어와 상품의 연결고리

### 정의
- 적합도 = 검색어와 상품이 얼마나 잘 맞는지의 점수
- **검색어에 따라 점수가 달라지는 유일한 점수**
- A로 검색했을 때와 B로 검색했을 때 순위가 뒤바뀌는 이유
- 네이버 공식: "인기도가 아무리 높더라도 상위에 랭크되지 않을 수 있다"

### 적합도의 2가지 점수 기준

#### ① 필드 연관도 (영역별 가점)
**가점 우선순위 (우측으로 갈수록 낮음)**:
```
★ 상품명 > 스토어명 > 브랜드, 제조사 > 카테고리, 태그, 속성
```
- 총 7가지 필드가 검색에 영향
- 검색어에 따라 모두 가점이 달라짐
- **공략 1순위: 상품명** (가장 강력)

#### ② 카테고리 선호도
- 검색어에 따라 특정 카테고리 선호가 다름
- 예: '휴지통' 검색 결과 vs '참치회' 검색 결과의 카테고리가 완전 다름
- 같은 '휴지통'도 [다용도 휴지통] vs [센서 휴지통]으로 카테고리 분리
- 키워드별 최적 카테고리를 사전에 검색해서 선택해야 함
- **카테고리 변경은 지옥** → 처음에 잘 골라야 함
- 예: '티셔츠' → 남성의류 62% / 여성의류 37% → 타겟 불분명 시 확률적으로 남성의류 유리

### 검색어마다 점수 다른 사례 — [커먼777]

- 같은 상품이 '대형 쓰레기통'에선 만점 1위 → '사무실 쓰레기통'에선 적합도 페널티로 2위
- 상품명에 '사무실'이 들어있어도 페널티 받음 (단순 포함 ≠ 만점)
- 만약 상품명을 「업소용 대형 쓰레기통 사무실 쓰레기통 카페 60리터」로 바꾸면 30분 뒤 1위 가능성
- 인기도 그대로 두고 적합도만 바꿔서 1위 달성 = 적합도의 위력
- **힌트**: 타겟 키워드끼리 붙여 쓰면 적합도는 만점 (더 이상 증가할 점수가 없음)

### 키워드 중복의 진실
- '쓰레기통' 카테고리에서는 쓰레기통을 3번 써도 페널티 없음
- 네이버는 중복 키워드를 쓰지 말라고 경고하지만, 실제 테스트 결과 페널티 없음
- "네이버 경고와 실제 동작이 다를 수 있다 — 테스트로 확인"

### 체급 전략 — 절대 1위 판매자 키워드 베끼지 마라
- "1위 판매자 키워드 그대로 베끼면 나도 상위노출 될까?" → **절대 안 됨**
- 갓 데뷔한 권투 초보가 챔피언과 붙으면 0%
- 핵심: '업로드하는 순간 상위권에 꽂히는 키워드'를 찾아내는 것

#### 사과 vs 홍옥 예시
- '사과' = 프로리그 (대형 키워드, 갓 데뷔는 안드로메다행)
- '홍옥' = 월 검색량 3,000건 알짜배기, 등록 즉시 1페이지 시작 가능
- "이길 수 있는 싸움터를 고르는 것, 그것이 로직의 첫걸음"

#### 보충
- '사과'라는 단어를 절대 쓰지 말라는 게 아님
- '청송부사사과' 같은 키워드를 쓰려면 '사과'는 어쩔 수 없이 들어감 → OK

### 배열의 마법 — 키워드끼리 손 잡기
- 네이버는 키워드 '배열'에 따라 추가 가점
- 검색어가 두 단어 이상일 때, 상품명에서 그 단어들이 얼마나 가까이/순서대로 붙어있는지가 중요
- 저자 표현: "키워드들이 손을 맞잡고 있다"

#### 상황 1 — '기저귀 휴지통' 검색 시
- 상품명 1번: 미니 **휴지통** 자동 가정용 **기저귀** 냄새 → 순위 하락 📉
- 상품명 2번: 미니 **기저귀 휴지통** 자동 가정용 냄새 → 순위 상승 🚀

#### 상황 2 — '기저귀 휴지통' 검색 시 (태그 활용은 안 됨)
- 상품명 1번: 미니 **휴지통** 자동 가정용 냄새 + 태그: #기저귀쓰레기통 → 순위 하락 📉
- 상품명 2번: 미니 **기저귀 휴지통** 자동 가정용 냄새 → 순위 상승 🚀
- 결론: 태그로는 부족, 상품명에 키워드를 손잡게 배치해야 함

### 위치(앞/뒤)의 진실 — 흔한 낭설 부수기
- "중요한 키워드는 무조건 맨 앞에 써야 유리하다?" → **거짓**
- 테스트 결과: 키워드가 맨 앞이든 중간이든 뒤든 검색 순위에 차이 없음
- 중요한 건 '위치'가 아니라 ① 키워드 포함 여부 ② 배열 관계

#### 상황 3 — '휴지통' 키워드로 노출하고 싶을 때
- 상품명 1번: **휴지통** 미니 기저귀 자동 가정용 냄새
- 상품명 2번: 미니 기저귀 자동 가정용 냄새 **휴지통**
- → '휴지통' 순위 동일

### 쓸모없는 키워드는 버려야 한다
- "예쁜 크리스마스 오르골" → 네이버 AI는 '예쁜'을 인식하지 않음, "크리스마스 오르골"만 인식
- **terms (텀즈)**: 네이버가 상품명을 분석할 때 끊어 읽는 실제 인식 단위
- 개발자 모드의 terms 항목을 보면 '예쁜' 같은 꾸밈말은 어디에도 없음
- 꾸밈말은 적합도에 영향 없이 글자 수만 차지 → 가점 0

### 적합도 총정리
- 적합도 = 키워드 선정 + 배열의 결과
- 카테고리/제목/태그 등에 어떤 키워드를 어디에 둘지 전략적으로 고민해야 함
- 적합도는 판매자가 노력으로 100% 개입할 수 있는 **유일한 영역**

## 1장 — ② 인기도 가점: 클릭과 구매로 증명하는 진짜 점수

### 정의
- 적합도가 '예선전'이라면 인기도는 '본선'
- **검색어와 무관하게 상품 자체에 부여되는 고정 점수**
- "이 상품은 잘 팔리는 상품인가?"의 절대적 채점표
- 네이버는 결국 수수료 플랫폼 → 매출 높은 셀러를 상위 노출 (자본주의 논리)

### 인기도 5가지 요소

| 요소 | 중요도 | 설명 |
|---|---|---|
| **판매실적** | ★★★★★ | 점수의 제왕. 최근 2일/7일/30일 판매금액·건수 지수화. 반드시 '네이버 쇼핑검색' 경유 구매여야 함 |
| **클릭수** | ★★★★ | 최근 7일 쇼핑검색 클릭 횟수 |
| **리뷰수** | ★ | 100개면 충분, 1,000개부터 만점 |
| **최신성** | ★ | 신규 업로드 버프, 시간 지나면 사라짐 |
| **찜수** | (무시) | 점수 무관, 시간 낭비 |

### 판매실적의 30일 법칙
- 판매실적은 평생 가지 않음
- **최대 30일이 유효기간**
- 31일 전 매출/판매개수는 31일째 0점으로 리셋
- 후발주자 기회: 한 달만 바짝 성과 내면 1위 자리 뺏을 수 있음
- 하루 이틀 반짝으로는 안 됨 → 일주일/한 달 단위 우상향 그래프가 핵심

### 사례 — 검색량 42,000 키워드 1위 등극
- 8월 8일: 적합도 최적화 (이후 상품명은 손대지 않음)
- 9월 8일: 한 달 후 1위 달성
- 인사이트: 적합도 세팅 후 클릭/판매가 자동 누적되며 인기도 점수 폭발

### 클릭수
- 최근 7일 동안 '쇼핑 검색'을 통해 내 상품을 클릭한 횟수
- 8일 전 기록은 초기화
- 클릭 어뷰징(조작) 업체: 점수에 도움 되긴 하지만 **권장하지 않음**
- 어뷰징 적발 시 영구 퇴출

### 리뷰수와 최신성
- **리뷰수**: 100개면 거의 만점, 1,000개부터 그냥 만점. 1만개여도 가산점 없음
- 다만 군중심리 효과는 무시 못함 (리뷰 많은순 필터 유입)
- **최신성**: 신규 상품 보너스, 시간이 지나면 점진적 감소 → 일정 시간 후 0점
- 6개월 된 상품 = 2년 된 상품 = 0점

### 찜수
- "찜 반사해요~" 시간 낭비
- 상위노출 로직과 무관

### 구매건수의 진실
- "구매건수 많으면 상위노출" → 반만 맞음
- 사례: [농부가된해적왕]은 구매 14,000건이지만 3위, 1,388건 판매자가 1위
- **구매건수 집계 방식 허점**:
  - 옵션 구매 → 1건 카운트
  - 추가상품 옵션 → 0건 카운트
- 네이버는 어뷰징 방지 위해 구매건수 단순 합산을 안 씀

#### 구매건수 카운트 예시
- 고객 1명이 옵션1(7개) + 옵션2(19개) + 추가상품(8개) 구매
- → 옵션1: 1건, 옵션2: 1건, 추가상품: 0건
- → 총 2건 카운트

### BEST 딱지의 정체
- "3일 데이터가 중요하다"는 말이 있지만 BEST 딱지는 단순히 최근 3일 판매 표시일 뿐
- 4일째 안 팔리면 사라짐
- 로직 점수와 무관 — 단지 시각적 장식
- **공식 로직은 판매실적(2/7/30일) + 클릭수(7일)** 둘 뿐

### 인기도 핵심 — 외부 유입 vs 쇼핑 검색

#### 결정적 사례
- 판매자 A: '휴대폰 케이스' 검색 매출 1,000만원 (외부 유입 0원)
- 판매자 B: '휴대폰 케이스' 검색 매출 500만원 + 유튜브 링크 매출 1,000만원 (총 1,500만원)
- → **상위 노출 승자: A**
- 이유: 네이버는 자기 검색을 통한 매출을 더 키워야 수수료를 챙김. B는 네이버 도움 없이 잘 팔고 있어 밀어줄 이유 없음

#### 네이버 공식 답변 (저자가 직접 상담사에 문의)
- 질문: "외부 플랫폼 URL 링크로 구매가 이뤄지면 인기도가 올라가나요?"
- 답변: "네이버 쇼핑을 통한 상품 클릭, 판매실적만 반영됩니다."
- **실전 팁**: 1명의 상담사가 아닌 **최소 3명의 상담사에게 문의**하는 것이 좋다. 상담사마다 경력이 달라 말이 다를 수 있기 때문이다. 캡처해두면 추후 문제 발생 시 증거로도 활용 가능.

#### 점수 인정 표 (캡처용)
| 유입 종류 | 클릭 점수 | 판매 점수 |
|---|---|---|
| 타 플랫폼 URL 링크 구매 | ❌ | ❌ |
| 네이버 쇼핑 광고 구매 | ❌ | ✅ |
| 네이버 파워링크 광고 | ❌ | ❌ |

- 네이버의 본질: 가두리 양식장 — 블로그/카페/지식iN/쇼핑까지 모든 행위를 네이버 안에서 끝내야 그들이 돈을 벎

### 인기도 점수 뻥튀기 4가지 전략

#### ① 링크(URL) 대신 '검색 키워드'를 줘라
- 외부 마케팅(유튜브/인스타/틱톡) 트래픽을 URL로 날리지 마라
- 고객에게 "네이버에 [OOO] 검색하세요"라고 유도
- 예: "로직해커엑스 마우스" 같은 (스토어명/브랜드 + 상품 키워드) 조합
- 검색해서 클릭 → 구매까지 가면 그 모든 과정이 인기도 점수
- 극단: 링크는 0점, 검색은 만점

#### ② 옵션 구성은 '그물'처럼 짜라
- 신발 사러 왔는데 215mm 없음 → 1초 만에 이탈 → 점수 하락
- 옵션을 촘촘하게 구성 → 구매 전환율 ↑ → 판매 실적 ↑

#### ③ 네이버 검색 광고를 영리하게 써먹어라
- CPC 광고는 클릭 점수는 못 받지만 **판매 실적 점수는 받음**
- 저자 광고 계정 평균 수익률 6,000%+ (객단가에 따라 다름)
- 핵심은 광고 수익률보다 세팅법

#### ④ 리뷰의 힘 (1점의 공포)
- 별점 1점 도배 vs 별점 5점 동영상 리뷰의 차이
- **저자 일화**:
  - 저자가 3위 시절, 2위 판매자가 1위 공격 제안
  - 저자는 거절했지만 2위는 실행 → 지인 동원해 1점 악성 리뷰 박제
  - 1위 판매자: 하루 100건 → 20건 (매출 80% 증발)
- 좋은 리뷰만큼 악성 리뷰 한 줄도 치명적

## 1장 — ③ 신뢰도 가점: 품질과 신뢰의 기본 베이스

### 정의
- 적합도/인기도가 점수를 '따는' 과목 → 신뢰도는 '지키는' 과목
- 거의 모든 셀러가 만점 받음 → 변별력 낮음
- 하지만 감점받으면 순위가 나락

### 체크 2가지

#### ① 상품명 SEO 점수 — '쇼핑 상품정보 검색품질 체크'
- 상품 등록 시 체크 버튼 존재
- 깨끗하면 만점
- 빨간불은 만점일 수도 있고 아닐 수도 있음 (예: 제조사/브랜드 미입력은 페널티 없음)
- 단, **허위 속성값 입력**은 상품 삭제 대상 → 수정해도 삭제 기간 동안 못 받은 인기도는 증발

#### ② 네이버 쇼핑 페널티
- 가구매, 가리뷰, 배송 지연 등 기본적 어뷰징만 안 하면 만점
- 결국 변별력 없음 → 승부는 적합도/인기도에서

### 50자의 법칙
- 네이버 기준은 50자
- **정확히는 49자까지만 안전**, 50자가 되는 순간 페널티
- 50자나 99자나 똑같이 1회 페널티
- 가능하면 50자 이내로 맞추는 게 안전
- 순위가 낮거나 신규 상품이면 무조건 50자 이내

### 30자 짧게 = 낭설
- "상품명은 30자 이내로 짧게 써야 노출이 잘 된다?" → **거짓**
- 50자 안에서는 글자 수가 적다고 가점 없음
- 넣을 키워드는 다 넣되 50자 안 넘기는 선에서 최적 길이 추구

### 예외 카테고리
- **에어졸/스프레이** 같은 특정 카테고리는 50자 초과해도 페널티 없음
- 1위 판매자가 50자 넘는 상품명 유지하는 사례 존재
- 하지만 절대 다수 카테고리는 50자 이내가 만점 기준

## 1장 — 적합도와 인기도의 무한 동력 (계단 오르기 모델)

### STEP 1 — 적합도 → 인기도 (시동 걸기)
- 100위 상품 vs 99위 상품 → 인기도 차이는 도토리 키 재기
- 적합도(상품명) 최적화 → 99위, 90위, 70위, 50위로 야금야금 상승
- 눈에 띄니까 클릭/판매 발생 → 인기도 점수 자연 상승
- 공식: **적합도 → 인기도**

### STEP 2 — 인기도 → 적합도 (체급 키우기)
- 인기도가 오르면 몸집이 커진 것
- 만족하지 말고 더 큰 키워드(중형)로 상품명 변경 → 새 적합도 최적화
- 동네 대회 → 도 대회로 진출
- "남의 상품명을 베끼는 게 아니라 철저하게 내 데이터와 내 체급에 맞는 키워드로 갈아타는 것"
- 공식: **인기도 → 적합도**

### 적합도는 검색어에 따라 다르다 — 마지막 강조
- '센서 휴지통' 검색 시:
  - 상품명 A: "센서 휴지통" → 만점
  - 상품명 B: "휴지통 센서" → 페널티
- 자리만 바꿔도 점수 달라짐 → 배열 전략의 기초

## 로직의 진화 — Linear → GBDT (네이버 변천사)

### 2022년 6월 23일 — 분기점
- 이 날 네이버는 검색 로직의 판을 완전히 뒤엎음
- 저자가 이 날짜를 강조하는 이유: "변화의 원리를 모르면 결국 지쳐서 포기하게 된다"

### 과거 — Linear (1차원적 계산기)
- 각 요소(Feature: 상품명/카테고리/클릭수/판매량 등)에 가중치 곱해서 단순 합산
- 적합도 + 인기도 + 신뢰도 단순 더하기
- '돈질'이 통하던 시대 → 마케팅으로 판매량만 키우면 상품명 엉성해도 1등 가능

### 현재 — GBDT (Gradient Boosting Decision Tree)
- 2022년 이후 도입된 깐깐한 AI 모델
- 핵심: **'조건부 채점'**
- 단순 합산이 아니라 소수점 단위로 쪼개서 조건문 적용

#### 조건문 예시
- "판매량이 높은가?" → Yes → "그럼 적합도는 높은가?" → No → **탈락 (-0.8점)**
- "판매량이 높은가?" → Yes → "그럼 적합도는 높은가?" → Yes → **합격 (+0.5점)**

### GBDT의 충격
- 과거: 판매량 높으면 무조건 점수
- 현재: 다른 조건(적합도) 안 맞으면 판매량 아무리 높아도 점수 깎임
- "이게 바로 당신이 죽어라 팔아도 순위가 안 오르는 진짜 이유"

### 네이버 공식 블로그 인용
> "Tree 구조를 이용하여 검색어와 상품 간의 매칭 정확도가 낮은 경우 상품의 인기도가 아무리 높더라도 상위에 랭크되지 않을 수 있습니다. 또한, 사용자 클릭 대비 리뷰나 판매지수가 높은 경우 더 상위에 랭크될 수도 있습니다."

### Linear → GBDT 전환 시 순위 역전 사례 (PDF 도표)
- Linear 모델에서 **1등**이었던 상품 '1'이 GBDT 모델 적용 후 상품 '5'에게 밀려 **3등으로 하락**
- 단순 합산이 아니라 복잡한 트리 구조를 통과하면서 순위가 뒤바뀜
- 결론: 같은 판매량이어도 적합도 조건문을 통과 못 하면 순위가 역전됨

### 증거 — 돼지고기 카테고리 사례
- [농부가된해적왕] 판매자: 구매 14,000건인데도 3위
- 1,388건 판매자가 1위
- GBDT가 적합도에서 점수 깎음 → 판매량으로 못 이김

### 최종 결론
- "많이 팔면 장땡"이라는 순진한 생각은 버려야 함
- GBDT 미로를 통과하는 유일한 열쇠 = **정교하게 설계된 적합도**

## 카테고리별 정리 (Writer/CS 응대용 빠른 참조)

### 키워드 전략
- 체급에 맞는 소형 키워드부터 1위 → 점진적 확장
- 핵심 1~2개에 집중 (분산 금지)
- 배열: 타겟 키워드끼리 붙여 쓰기
- 위치는 무관, 배열만 중요
- 꾸밈말('예쁜', '귀여운') 등은 텀즈 인식 안 됨

### 상품명
- 50자 이내 (49자까지 안전)
- 같은 키워드 중복 가능 (카테고리 따라)
- 카테고리 사전 검색 후 선택

### 인기도 올리는 법
- 외부 마케팅 → 검색 키워드로 유도
- 옵션 풍부하게
- 네이버 쇼핑 광고는 판매 점수는 받음 (클릭 점수는 X)
- 리뷰 100개 이상

### 절대 금지
- 외부 링크 직접 유입 (인기도 0점)
- 클릭 어뷰징 (영구 퇴출)
- 가구매/가리뷰 (페널티)
- 허위 속성 입력 (상품 삭제)
- 50자 초과 상품명 (대부분 카테고리)
