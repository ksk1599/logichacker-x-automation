# 2강 — 구조 설계(1): 캠페인(Campaign)과 그룹(Group)의 위계질서

## 핵심 메시지
- 광고를 돌리기 전에 지휘 체계(위계질서)부터 이해해야 한다
- 캠페인 > 그룹 > 소재, 이 3단계 구조를 모르면 데이터 분석이 불가능하다
- 각 단계에서 설정 가능한 옵션이 다르다 — 어디서 무엇을 조절하는지 알아야 한다

## 3단계 구조 (폴더 비유)

```
📁 광고 캠페인  ← 윈도우 탐색기의 '최상위 폴더'
  📂 광고 그룹   ← '하위 폴더'
    📄 광고 소재  ← '개별 파일'
```

- 하나의 캠페인 안에 여러 그룹
- 하나의 그룹 안에 여러 소재

## 단계별 설정 가능 항목

### (1) 광고 캠페인 — 전략의 큰 그림
- **하루 예산**: 하루에 쓸 돈의 상한선 (소진되면 자동 OFF)
- **노출 기간**: 광고를 켤 기간 설정 (시간대 설정은 그룹에서 가능)

### (2) 광고 그룹 — 가장 세밀한 전술을 짜는 핵심 단계
- **기본 입찰가**: 클릭당 지불할 의향이 있는 최대 금액
- **매체 설정**: PC/모바일/파트너 매체 노출 여부
- **입찰 가중치**: PC와 모바일에 서로 다른 가격표 적용 가능
- **확장 소재**: 네이버 톡톡, 추가 홍보 문구 등
- **제외 검색어**: 특정 키워드에서 광고가 뜨지 않게 막는 방패 (핵심)

### (3) 광고 소재 — 실제 전장에 나가는 병사
- 실제 상품 등록 및 노출용 상품명/이미지 설정
- 소재별 입찰가 개별 설정 가능

## 11가지 설정 충돌 퀴즈 (자가진단)

> 아래 퀴즈를 풀 수 없다면 광고비를 효율적으로 쓸 준비가 안 된 것이다.
> 정답은 ch03에서 공개.

**설정 구조:**
```
📁 A 광고 캠페인 (최상위)
  📂 A-1 광고 그룹 (중간)
    📄 A-1-1 광고 소재 (최하위)
```

| 번호 | 질문 |
|---|---|
| Q1 | A 캠페인 예산 10만 원 vs A-1 그룹 예산 5만 원 → 각각 최대 얼마? |
| Q2 | A 캠페인 예산 5만 원 vs A-1 그룹 예산 10만 원 → 각각 최대 얼마? |
| Q3 | 그룹 입찰가 500원 vs 소재 입찰가 300원 → 실제 입찰가는? |
| Q4 | 그룹 입찰가 300원 vs 소재 입찰가 500원 → 실제 입찰가는? |
| Q5 | 그룹 문구 "50매 무료배송" vs 소재 문구 "100매 무료배송" → 어떤 문구 노출? |
| Q6 | 그룹 문구 없음 vs 소재 문구 "100매 무료배송" → 어떤 문구 노출? |
| Q7 | 그룹 제외 {A,B,C,D,E} + 소재 제외 {D,E,F,G,H} → 최종 제외 키워드는? |
| Q8 | 그룹 70개 MAX + 소재 70개 MAX → 총 몇 개 제외 가능? |
| Q9 | 그룹: 검색 네트워크 노출 제외 ON / 소재: OFF → 결과는? |
| Q10 | 그룹: 검색 네트워크 노출 제외 OFF / 소재: ON → 결과는? |
| Q11 | 동일 상품 ID로 소재 15개(10 ON, 5 OFF) → 정상 노출되는가? |

## 인용 가능한 문장
- "캠페인 > 그룹 > 소재. 이 위계질서를 머릿속에 각인해라. 이것을 모르면 나중에 데이터가 뒤섞여 분석 자체가 불가능해진다."
