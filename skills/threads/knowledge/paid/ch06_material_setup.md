# 6강 — 실전 세팅(3): 소재 등록 및 그룹 확장 테크닉

## 핵심 메시지
- 소재는 고객이 가장 먼저 마주하는 '얼굴' — 대충 상품명 복사하면 클릭률에서 진다
- 소재마다 다른 노출용 상품명과 이미지를 설정하는 것이 차별점
- 제외 키워드 '일치' 옵션 + '순서 엄수' 로직을 이해하고 140개 방패를 채워라

---

## 소재 등록 흐름

1. 광고 관리자에서 '검색하기' 버튼 → 내 상품 목록 표시
2. 광고할 상품 선택 → '광고 만들기' 클릭
3. 소재 검토 화면 확인

> ⚠️ 헷갈림 주의: '소재' 탭은 광고 소재, '확장 소재'·'제외 검색어'는 광고 그룹 탭

---

## 1. 노출용 상품명 & 이미지 (핵심 차별화)

소재 내 '상세보기' → '수정' 버튼에서 설정

**핵심**: 소재마다 **다른 상품명 + 다른 썸네일** 적용 가능

- 예시: 빨간 안경, 파란 안경 옵션이 있을 때
  - 빨간 안경 소재 → 노출용 상품명: 빨간 안경 관련 키워드 / 이미지: 빨간 안경 썸네일
  - 파란 안경 소재 → 노출용 상품명: 파란 안경 관련 키워드 / 이미지: 파란 안경 썸네일

**왜 차별점이 되는가?**
- 대부분의 셀러는 1 광고그룹 : 10 소재를 제대로 세팅하지 않음
- 이를 제대로 세팅하는 셀러가 거의 없음 → 여기서 남들과 다른 차별점 창출

---

## 2. 확장 소재 설정

### 추가홍보문구
- 로직(검색인식)에 영향 없음 (검색 키워드로 인식 안 됨)
- 고객에게 후킹이 될 만한 요소 기재 (이벤트, 혜택 등)
- 고급 옵션의 요일/시간/기간 설정은 기본값 유지

### 쇼핑 상품 부가 정보
- 활성화 시: 구매건수, 리뷰수, 평점, 찜수 노출
- 비활성화 시: 위 항목들이 노출되지 않음
- **필수 활성화 권장**

### 네이버 톡톡
- 확장 소재 → 저장 후 닫기 (별다른 설정 불필요)
- 설정 시 톡톡 버튼 활성화됨

---

## 3. 제외 검색어 (소재 단위) — 140개 방어막

### 검색 네트워크 영역 설정
- **'사용'** 클릭: 카테고리 클릭 시 나오는 광고 지면에서 노출 안 됨
- **'사용 안 함'** (기본값): 해당 지면에도 광고 노출됨
- **우리 세팅**: '사용' 선택 (검색을 통한 구매가 목적 + 직접 관리)

### 제외 검색어 유형 선택

| 유형 | 의미 | 예시 ('안경' 제외 등록 시) |
|---|---|---|
| **일치** (권장) | 등록 키워드와 완전 일치할 때만 제외 | '안경' 제외, '노란 안경' '레이벤 안경'은 노출 O |
| 구문 | 등록 키워드가 구문으로 포함되면 모두 제외 | '안경' 포함된 모든 키워드 제외 (띄어쓰기 무관) |

**우리는 '일치' 옵션 사용**

### 140개 방어막 전략

| 위치 | 제외 가능 수 |
|---|---|
| 광고 소재 | 최대 70개 |
| 광고 그룹 | 최대 70개 |
| 합계 | 최대 **140개** |

**핵심**: 소재와 그룹에 같은 키워드 중복 등록 금지 → 서로 다른 키워드로 140개 완성

**왜 1:1 매칭 전략이 필요한가?**
- 1개 그룹에 여러 키워드 관리 시:
  - A 키워드(1위에 1,000원 필요) + B 키워드(1위에 100원이면 충분) → 둘 다 1,000원 적용
  - B 키워드에 불필요한 광고비 낭비
- **해결**: 1그룹 1키워드 + 140개 제외 키워드 → 키워드별 최적 입찰가 설정 가능

---

## 요약

1. 소재는 가면이다: 10개의 노출용 상품명으로 다양한 검색 의도를 잡아라
2. 제외 키워드는 과학이다: '일치' 옵션과 순서 엄수 로직을 이해하고 140개 방패를 채워라
3. 쇼핑 상품 부가 정보 반드시 활성화
4. 확장 소재에서 후킹 문구 세팅
