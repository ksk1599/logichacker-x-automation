# Writer Agent — 콘텐츠 정확성 규칙

> 글 작성 전 이 파일을 읽어라.
> 도메인 오류가 담긴 글은 QA에서 반드시 반려된다.

---

## 작성 전 필수 체크

글에 스마트스토어 알고리즘 관련 주장이 들어간다면:
1. Knowledge Agent에 `tier=publishable`로 확인 요청
2. 이 파일의 금지 목록과 대조

---

## 절대 쓰면 안 되는 주장

### ❌ 키워드 위치 가점

다음 내용은 사실이 아니다. 절대 글에 쓰지 마라.

```
❌ "앞에 올수록 가점 N배"
❌ "앞에 온 키워드에 2배 이상 가점"
❌ "중요 키워드는 맨 앞에"
❌ "키워드 배열 순서가 검색량 반영도를 결정"  ← 이 표현도 금지
```

**올바른 대체 표현:**
```
✅ "키워드끼리 가까이 붙어야 한다"
✅ "배열 관계가 가점을 결정한다"
✅ "손 맞잡기 — 키워드를 붙여 써야 가점"
```

## 용어 규칙

- "가중치" → "가점"으로 표현
- "검색 알고리즘 가중치" → "검색 알고리즘 가점" 또는 "검색 점수"
